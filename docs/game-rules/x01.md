# X01 (All Bulls = 50)

## Objective
Reduce your score from the starting value to exactly 0.

Common starting values:
- 301
- 501
- 701

## Turn structure
- Each player throws up to 3 darts per turn.
- The score of each dart is subtracted from the player’s remaining score.

## Scoring
- Single ring = face value
- Double ring = face value × 2
- Triple ring = face value × 3
- Bullseye = 50

There is **no 25-point bull**.

Examples:
- S20 = 20
- D20 = 40
- T20 = 60
- Bull = 50

## Basic win condition
A player wins by reaching **exactly 0**.

## Bust rule
A bust occurs if a player:
- goes below 0, or
- reaches 1 remaining (with double-out or master-out enabled)
- reaches 0 on an invalid finish (e.g. single with double-out, or single with master-out)

When a bust happens:
- the entire turn is canceled
- the score resets to what it was at the start of the turn
- the turn ends immediately

## Double-out variant
In double-out games:
- the final dart must be a double to win
- the bull (50) counts as a double

Examples:
- 40 remaining → D20 wins
- 32 remaining → D16 wins
- 50 remaining → Bull wins
- 20 remaining → S20 → bust (not a double)

## Master-out variant
In master-out games:
- the final dart must be a double, triple, or bull to win
- single darts cannot finish the game

Examples:
- 40 remaining → D20 wins
- 60 remaining → T20 wins
- 50 remaining → Bull wins
- 20 remaining → S20 → bust (not a double, triple, or bull)
- 18 remaining → T6 wins
- 18 remaining → D9 wins

Note: master-out and double-out are mutually exclusive options. When master-out is enabled, triples are valid finishes in addition to doubles and bull.

## Double-in variant
In double-in games:
- a player must hit a double before scoring begins
- darts thrown before opening score 0
- once opened, scoring works normally

## Per-dart logic
For each dart:
1. Determine the dart value.
2. Subtract it from the current score.
3. If score < 0 → bust.
4. If score == 1 → bust.
5. If score == 0:
    - single-out → win
    - double-out → win only if the dart was a double or bull
    - master-out → win only if the dart was a double, triple, or bull
6. Otherwise continue until 3 darts are thrown or the turn ends.
