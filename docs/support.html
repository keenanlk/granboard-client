Support

Need help with NLC Darts? We use GitHub Issues to track bugs and feature requests.

Report a Bug

If something isn't working correctly, please open an issue and include:

  • What you expected to happen
  • What actually happened
  • Your device and iOS version
  • Your GranBoard model (if relevant)

Request a Feature

Have an idea for a new game mode or improvement? We'd love to hear it. Open an issue describing what you'd like to see.

Open an Issue on GitHub

Compatibility

NLC Darts is designed to work with GranBoard dartboards that support Bluetooth connectivity. If you're having trouble connecting, make sure Bluetooth is enabled and your board is powered on and in range.
